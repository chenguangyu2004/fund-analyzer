# 基金分析App

基于Python开发的基金分析工具，支持实时获取基金数据、计算盈亏、分析买卖策略。

**版本**: v5.2.0 (2026-03-21)
**特点**: 无需注册、完全免费、数据全面、智能分析

---

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动应用


## 使用说明

### 基本操作
1. 在页面输入6位基金代码（如：000001）
2. 输入持仓成本（元/份）
3. 输入持有份额
4. 点击"分析基金"按钮
5. 查看结果：
   - 基金基本信息
   - **双净值对比**（官方收盘净值 vs 实时估算净值）
   - 盈亏计算
   - **持仓股票实时价格和涨跌幅** ✨
   - 买卖策略建议
   - 历史净值走势

### 净值数据说明

#### 官方收盘净值（🟡）
- 基金公司每日公布的正式净值
- 更新时间：交易日收盘后（通常15:00后）
- 数据来源：东方财富网
- 用途：用于正式结算、申购赎回

#### 实时估算净值（🟢）
- 基于基金持仓股票的实时价格估算
- 更新时间：交易时间实时更新（工作日9:00-15:00）
- 数据来源：天天基金网
- 用途：盘中参考、了解实时盈亏变化
- **注意**：估算值仅供参考，不作为交易依据

### 净值对比说明
当同时获取到官方和实时数据时，页面会显示：
- 官方收盘净值（黄色背景框）
- 实时估算净值（绿色背景框）
- 实时估算与官方净值的差异百分比

### 自动刷新功能
1. 分析基金后，勾选"自动刷新数据（交易时间有效）"
2. 选择刷新间隔：30秒、1分钟、2分钟、5分钟
3. 系统会自动更新数据
4. 适合交易时间关注基金表现

### 保存基金功能
1. 输入基金信息后，点击"💾 保存基金"按钮
2. 已保存的基金显示在页面顶部
3. 点击基金卡片可快速填入信息
4. 点击"刷新"可立即获取最新数据
5. 点击"删除"可移除该基金

**注意**: 已保存的基金存储在浏览器本地，清除浏览器数据会丢失。

## 常见问题

### Q: 需要注册账号吗？
A: 不需要！所有数据来自公开渠道，无需注册、无需配置任何Key。

### Q: 提示"无法获取基金信息"怎么办？
A: 请检查：
- 基金代码是否正确（6位数字）
- 网络连接是否正常
- 该基金是否为在售的公募基金

### Q: 为什么有时候没有实时估算净值？
A: 实时估算净值仅在交易时间（工作日9:00-15:00）提供，非交易时间显示官方收盘净值。

### Q: 实时估算净值准确吗？
A: 实时估算净值基于持仓股票的实时价格计算，仅供参考：
- 可能存在一定误差（股票交易与基金结算的时差）
- 不作为申购赎回的依据
- 最终以基金公司公布的官方净值为准

### Q: 持仓股票价格显示"--"是什么意思？
A: 说明该股票的实时价格暂时无法获取，可能原因：
- 股票停牌
- API请求失败
- 数据源临时不可用
- 可点击股票代码查看详情

### Q: 港股价格支持吗？
A: 支持！应用已实现港股（5位代码）的实时价格获取，例如：
- 00700 腾讯控股
- 00941 中国移动
- 02318 中国平安

### Q: 实时估算与官方净值的差异说明了什么？
A: 差异反映了盘中市场变化：
- 正差异：市场上涨，估值高于收盘价
- 负差异：市场下跌，估值低于收盘价
- 差异大小反映当日市场波动程度

### Q: 支持哪些类型的基金？
A: 支持所有公开交易的公募基金（股票型、混合型、债券型、指数型等）

### Q: 数据更新频率？
A:
- 官方净值：每个交易日更新一次，收盘后公布
- 实时估算：交易时间内实时更新
- 自动刷新：可设置30秒-5分钟

### Q: 什么时候使用自动刷新？
A: 适合以下场景：
- 交易日需要关注盘中变化
- 短线操作需要及时数据
- 监控多个基金的实时表现

### Q: 如何修改端口？
A: 编辑 `app.py` 文件，修改最后一行的端口号：
```python
app.run(debug=True, host='127.0.0.1', port=5000)  # 修改5000为其他端口
```

### Q: 已保存的基金会丢失吗？
A: 数据保存在浏览器本地，只有清除浏览器数据或使用隐私模式时才会丢失。

### Q: 修改代码后如何应用？
A: 运行以下命令重启服务器：
```bash
# Windows
start.bat restart

# Linux/Mac
./start.sh restart
```
