# 新功能说明

## v5.0.0 更新内容 (2026-03-20)

### 🆕 实时更新功能

**实现原理：**
- 在交易时间（工作日9:00-15:00），系统会自动获取基金的实时估算净值
- 实时估算净值基于天天基金网提供的持仓股票实时价格计算
- 非交易时间显示官方收盘净值

**数据来源：**
- 官方净值：东方财富基金数据
- 实时估算：天天基金网实时估算API (http://fundgz.1234567.com.cn)

**显示标识：**
- 🟢 实时数据：绿色标签，表示基于持仓股票的实时估算
- 🟡 收盘数据：黄色标签，表示官方公布的收盘净值

### 🆕 自动刷新功能

**功能说明：**
- 勾选"自动刷新数据"后，页面会自动定时更新基金数据
- 可选择刷新间隔：30秒、1分钟、2分钟、5分钟
- 仅在交易时间有效，避免非交易时间无效请求

**使用方法：**
1. 分析基金后，页面底部会显示自动刷新控制栏
2. 勾选"自动刷新数据（交易时间有效）"
3. 选择刷新间隔
4. 系统会自动更新数据，无需手动刷新

**注意事项：**
- 自动刷新仅在交易时间（9:00-15:00）有意义
- 频繁刷新会增加服务器负载，建议不要选择过短间隔
- 实时估算是基于持仓股票的估算值，可能存在误差

### 🆕 数据更新时间

页面会显示数据更新时间，方便用户了解数据的时效性：
- `HH:MM:SS` 格式的最后更新时间
- 数据来源说明（实时/收盘）

## 技术实现

### 后端改动
1. `FundAnalyzer.get_fund_info()` 方法增强：
   - 添加实时数据获取逻辑
   - 判断当前是否为交易时间
   - 合并官方净值和实时估算数据

2. 新增 `FundAnalyzer._get_realtime_estimate()` 方法：
   - 从天天基金网获取实时估算净值
   - 返回估算净值和增长率

### 前端改动
1. 添加自动刷新控制UI
2. 新增 `toggleAutoRefresh()` 和相关函数
3. 显示数据来源标识（实时/收盘）
4. 添加更新时间显示

## 使用场景

### 适合使用实时更新的场景：
- 交易日盘中需要关注基金表现
- 需要及时了解盈亏变化
- 短线操作需要实时数据

### 不适合使用实时更新的场景：
- 长期投资者关注长期趋势
- 非交易时间查看
- 数据流量有限的情况
