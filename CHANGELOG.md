# 基金分析器更新说明

## 最新更改
已将数据源从东方财富切换到天天基金网API

## 数据源说明
- 实时数据: http://fundgz.1234567.com.cn/js/{基金代码}.js
- 历史数据: http://fund.eastmoney.com/f10/F10DataApi.aspx

## 使用方法
1. 安装依赖: `pip install -r requirements.txt`
2. 运行应用: `python app.py` 或双击 `start.bat`
3. 打开浏览器: http://127.0.0.1:5000

## 输入说明
- 基金代码: 6位数字，例如 000001
- 持仓成本: 您购买时的净值价格
- 持有份额: 您持有的基金份额数量

## 注意事项
- 天天基金网API是免费公开接口，可能有访问频率限制
- 如遇到数据获取失败，请稍后重试
- 建议使用真实的基金代码进行测试
