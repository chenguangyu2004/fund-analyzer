# 基金分析App 使用说明

## 快速开始

### Windows用户
双击运行 `start.bat` 文件即可启动应用

### Linux/Mac用户
```bash
chmod +x start.sh
./start.sh
```

### 手动启动
```bash
pip install -r requirements.txt
python app.py
```

然后打开浏览器访问：http://127.0.0.1:5000

## 功能说明

### 1. 基金信息查询
- 输入6位基金代码（如：000001）
- 自动获取基金名称、类型、净值等基本信息

### 2. 盈亏计算
- 输入持有的基金总额
- 自动计算当日盈亏金额和比例
- 可选输入买入成本，计算总盈亏

### 3. 策略分析
- 基于历史净值数据分析买卖建议
- 提供风险等级评估
- 显示分析信心度

### 4. 历史走势
- 显示近30天净值走势图
- 支持交互式查看

## 数据来源

本应用使用 AkShare 库获取基金数据，数据来源于东方财富网公开信息。

## 注意事项

1. **数据时效性**：基金净值数据可能有延迟，建议在交易日查看
2. **投资风险**：策略分析仅供参考，投资有风险，入市需谨慎
3. **网络连接**：需要网络连接才能获取实时数据
4. **基金代码**：请确保输入正确的6位基金代码

## 常见问题

### Q: 提示"无法获取基金信息"怎么办？
A: 请检查：
- 基金代码是否正确（6位数字）
- 网络连接是否正常
- 该基金是否为在售基金

### Q: 策略建议的准确度如何？
A: 策略分析基于历史数据统计模型，仅供参考，不构成投资建议。实际投资还需考虑个人风险承受能力、市场环境等因素。

### Q: 如何修改端口？
A: 编辑 `app.py` 文件，修改最后一行的端口号：
```python
app.run(debug=True, host='127.0.0.1', port=5000)  # 修改5000为其他端口
```

### Q: 支持哪些类型的基金？
A: 支持东方财富网上可以查询到的所有公开基金（股票型、债券型、混合型、货币型等）

## 技术支持

如遇到问题，请检查：
1. Python版本 >= 3.8
2. 所有依赖是否正确安装
3. 端口5000是否被占用
